# TMUX Session Theming - Quick Reference

## Available Themes

| Theme | Color | Best For |
|-------|-------|----------|
| `cobalt` | Deep blue/gray | Default, general use |
| `green` | Mossy green | Coding, development, AI sessions |
| `blue` | Soft blue | Research, learning, documentation |
| `lblue` | Bright light blue | Light background terminals |
| `lgreen` | Bright light green | Light background terminals |
| `lorange` | Bright light orange | Light background terminals |
| `lred` | Bright light red | Light background terminals |
| `purple` | Muted purple | Creative work, writing, notes |
| `orange` | Soft orange | Testing, debugging |
| `red` | Muted red | Production, urgent tasks |
| `nord` | Arctic blue | Clean, minimal aesthetic |
| `everforest` | Forest green | Nature-inspired, calm coding |
| `gruvbox` | Warm retro | Retro terminal lovers |
| `cream` | Warm neutral | Light backgrounds |
| `gray` | Neutral gray | Monochrome setups |
| `lightgray` | Light gray | Light backgrounds |
| `adaptive` | Auto-detect | Works on light & dark |

## Light Theme Patterns

The auto-detection for light themes (lblue, lgreen, lorange, lred) matches these keywords:

- **lblue**: `lblue`, `lightblue`
- **lgreen**: `lgreen`, `lightgreen`
- **lorange**: `lorange`, `lightorange`
- **lred**: `lred`, `lightred`
